# 🏠 Leadera
### CRM de seguimiento de leads para agentes inmobiliarios

Leadera es una herramienta pensada para agentes inmobiliarios que necesitan organizar su flujo diario de trabajo: saber con quién contactarse hoy, registrar cada interacción con un lead y hacer seguimiento de toda la cartera desde un solo lugar.

## ✨ Funcionalidades

- 🔐 **Autenticación** — registro e inicio de sesión de agentes
- 📋 **Home diario** — leads calientes, nuevos leads y leads para seguimiento del día
- ✅ **Gestión del día** — una vez registrada una interacción, el lead pasa a esta sección
- 🔍 **Cartera completa** — listado de todos los leads con filtros por estado y buscador
- 👤 **Detalle del lead** — datos personales, estado, búsqueda e historial de interacciones
- ➕ **Creación de leads** — alta de nuevos leads con búsqueda asociada

## 🛠 Stack tecnológico

**Backend:** Java · Spring Boot · Spring Security · JPA  
**Frontend:** Angular · TypeScript · HTML · CSS
